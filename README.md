1. **Clone the repository** (or unzip the provided project):

   git clone https://github.com/shreyasudaya/websocket-chat

   cd websocket-app

2. **Install backend dependencies:**

    cd backend

    npm install

3. **Install frontend dependencies:**

    cd frontend

    npm install

4. **Start Backend**

    cd backend

    npm start

5. **Run Frontend**

    cd frontend

    npm run dev
